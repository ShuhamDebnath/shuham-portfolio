# Shuham Debnath — App Studio Portfolio

This is the source code for the personal developer portfolio website of Shuham Debnath.

## Overview
A modern, privacy-first, lightweight portfolio website built entirely with plain HTML, CSS, and Vanilla JavaScript. The site showcases the app studio's mission, highlights released apps like CareLog, and provides terms and privacy policies.

## Features
- **Ultra-Fast Performance:** Zero heavy frameworks (no React, no Vue, no Tailwind dependencies). Built with pure HTML, CSS, and JS.
- **Pure CSS/HTML Device Mockup:** The hero section features a fully coded iPhone mockup (no raster images), resulting in a tiny footprint and crisp rendering on all displays.
- **Fluid Typography:** Uses `clamp()` functions to ensure typography scales perfectly across all screen sizes.
- **Responsive Design:** Mobile-first layout that elegantly adapts to tablets and desktop screens.
- **Dark Mode Aesthetic:** Sleek "Aurora" styling with dynamic, subtle animations.

## Repository Structure
- `index.html` - The main landing page showcasing the portfolio.
- `privacy.html` - The studio's privacy policy.
- `terms.html` - The terms of service.
- `styles.css` - All styling, custom properties, and animations.
- `script.js` - Minimal vanilla JavaScript for scroll-spy, navigation toggles, and dynamic dates.
- `favicon.svg` - Scalable vector favicon.
- `assets/` - Directory for any remaining static image assets.

## Deployment
This is a static website. It can be deployed instantly to any static hosting provider such as:
- GitHub Pages
- Cloudflare Pages
- Vercel
- Netlify

Simply deploy the root directory. No build step is required!
